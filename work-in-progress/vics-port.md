# VICS port — text editor from Medli C# to Makar C

> **Status:** placeholder — implementation not yet started
> **Milestone:** medium-term

## Summary

Port the VICS text editor from the Medli C# implementation to a Makar C
implementation, keeping the same key bindings and file format so users
have a consistent editing experience on both OSes.

## Acceptance criteria

- [ ] Open, edit, and save text files on the FAT32 volume
- [ ] Key bindings identical to the Medli VICS implementation
- [ ] File format compatible with the Medli VICS file format
- [ ] Works over both VGA/VESA terminal and serial console
